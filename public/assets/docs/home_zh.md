# 欢迎来到 RCV 实验室

机器人与计算机视觉实验室（RCV Lab）致力于机器人领域的前沿研究。实验室侧重于同步定位与建图（SLAM）、语义场景理解和人机交互等关键技术的开发和应用。

我们的研究涵盖了机器人自主导航、环境感知、智能决策等多个方面，旨在推动机器人技术在现实世界中的广泛应用。

RCV 实验室是深圳市机器人视觉与导航重点实验室（筹）的所在地。该重点实验室位于中国深圳的南方科技大学（SUSTech）电子与电气工程系，由南方科技大学讲席教授张宏领导。张宏教授在海外学术界工作 30 多年归国后，于 2020 年创立了 RCV 实验室。RCV 实验室目前支持其 25 多名成员（包括博士后研究员、博士生、硕士生以及研究助理）开展研究。实验室配备了各种机器人设备和传感器，并获得了来自公共和私人来源的充足资助。

# 当前研究项目

我们对使用视觉作为机器人感知环境、创建表示形式和做出智能决策的主要信息来源特别感兴趣。同时，我们还研究其他形式的传感，包括可以补充视觉传感的激光雷达（LiDAR）和偏振成像。

当前的研究项目包括：
- **同步定位与建图 (SLAM)**
- **语义场景理解**
- **人机交互**
- **偏振相机感知**

有关研究项目的更多详细信息，请参阅研究页面。
